1. Xây dựng trang chủ bán hàng loại sản phẩm giày sử dụng Tailwind CSS (https://tailwindcss.com) hoặc các Template tương đương. Khi đăng nhập thành công thì vào trang chủ và hiển thị trang chủ với những thông tin khuyến mãi, mới nhất, bán chạy nhất,....

3. Xây dựng chức năng tìm kiếm và lọc dữ liệu với nhiều điều kiện lọc.
